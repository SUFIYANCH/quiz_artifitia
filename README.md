# Quiz Artifitia

A simple quiz app built with Flutter and using Riverpod for state management.

## Getting Started

1. **Clone the repository:**

   ```bash
   git clone https://github.com/SUFIYANCH/quiz_artifitia.git
   ```

2. **Install dependencies:**

   ```bash
   flutter pub get
   ```

3. **Run the app:**

   ```bash
   flutter run
   ```

- **Model**
- **Providers**
- **Services**
- **Utils**
- **View**
